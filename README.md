# Smart Tele-Controlled IV System for Reverse Blood Flow Prevention and Automated Drug Delivery

## Overview
This project presents an IoT-based smart tele-controlled IV system for reverse blood flow prevention and automated infusion pump-based drug delivery. The system is designed to improve patient safety during IV therapy by combining real-time monitoring, optical backflow detection, and automated medication delivery through a web-based remote monitoring platform.

## Key Features
- Reverse blood flow detection using LED–LDR optical sensing
- Automatic solenoid valve control for backflow prevention
- Automated infusion pump for controlled drug delivery
- Real-time monitoring of:
  - Heart Rate
  - SpO₂
  - Body Temperature
- Web-based dashboard for remote monitoring and control
- ESP32-based wireless communication
- Low-cost and portable biomedical device design

## Outcome
The developed system successfully demonstrated:
- Reverse blood flow detection accuracy of 85%
- Automated valve response for backflow prevention
- Average infusion flow rate of 0.83 mL/sec
- Remote monitoring and control through a web dashboard
